#INSTALLATION INSTRUCTIONS 
1. Clone or Download the repository
2. Run npm install to install all dependencies
3. Run npm start to start the app
4. Access endpoints on localhost:9001

#REQUIREMENTS

Node v10.15.0 or above
Npm v6.4 or above
